Java 17+

Spring Data JPA

Spring Security (للحماية)

 تشغيل التطبيق محليًا
 تأكد من تثبيت Java وMySQL.

قم بتحديث application.properties لإعداد قاعدة البيانات.

استخدم mvn spring-boot:run لتشغيل التطبيق.

استخدم Postman أو Swagger لاختبار الـ API.
